A. Client

The Grid is part of my Personal Portfolio and it will be used to display the blog posts.

Initial Functionality:

  1. Archive
  2. Large number of posts Get access to hundreds of posts, efficiently
  3. Easy access To be able to get a glimpse of 20-30 blog posts in a single view.
  4. Readable
  5. Simplify displayed data Get rid of low importance information and show only the basic data points.
  6. Responsive
  7. Mobile/Tablet Work on Small devices.
B. Challenge

The challenges were distilled to 3 primary areas to focus on:

  1. Archive large number of posts
  2. Simplify displayed data
  3. Create Responsive Content for Tablets and Mobile Devices
C. Approach

Here we take a look at how those top problems were solved.

  1. Archive large number of posts

    GRID

    A grid with columns will solve the problem of displaying many articles and it is easy to make it responsive afterwards.

  2. Simplify displayed data

    Title, date, post number

    Keeping the essentials, only 3 data points remained on the table. The title of the post, the publishing date, and the post number that is used for displaying the articles in order and to make them easy to identify.

  3. Responsive Content - Mobile/Tablet

    Bootstrap Framework [custom breakpoints for different screen sizes]

    Multiple breakpoints to determine the exact width of the screen at which to perform the changes. A new set of values will be applied for the column width, to reduce the wide 4 column grid to 3, 2, and eventually 1 column as the screen size is getting smaller.

  4. Additional

    Typography - Google Fonts

    'Quattrocento' and 'Quattrocento Sans' were the result of testing multiple typeface combinations provided by Google Fonts.

The serif font choice is based on allowing the reader to get through as many titles as possible with a fast tempo, alongside with the font size choice.

Transitions

A small effect on hovering the post title can be observed. The scope of this is to ease the user interaction with the software. If it gets unnoticed, it is for the better as any interference between the person and the platform will minimize it's purpose.

Colors

The color choice is a soft combination of blue and green for hovering the title, and 2 grey colors: one stronger for title, and one lighter for the publishing date.
